# 🚗 Barabilar - The Ultimate Car Marketplace

Barabilar is an online platform that enables individuals and businesses to **buy, sell, swap, and auction cars**. The platform also supports businesses offering car-related services, including **spare parts sales** and **paid advertisements** to promote listings. Businesses are charged **50 kron per post**.

---

## 📌 Project Scope

### 🔹 **Key Features**

#### 👤 **User Management**
- User Registration (Individuals & Businesses)
- Login/Authentication (Email, Phone, Social Login)
- Profile Management
- KYC Verification (Government-issued ID Upload)
- Password Reset & Account Recovery

#### 🚘 **Car Listings & Marketplace**
- Add, Edit, and Delete Car Listings
- Buy, Sell, and Swap Cars
- Vehicle Auctioning System
- Business Accounts for Car Dealers and Parts Sellers
- Location-Based Search (**Nation, Country, State, Local Area, Landmarks, Community**)
- Image Upload & Management for Listings
- Pricing & Negotiation System with In-App Chat

#### 📢 **Advertising & Promotions**
- Paid Advertisements for Car Listings & Spare Parts
- Ad Packages with Different Pricing Tiers
- Payment per Post (**50 kron**)
- Secure Payment Integration (**Card, Bank Transfer**)

#### 🔍 **Search & Filters**
- GPS-Based Search System
- Category-Based Search System
- Advanced Filters (**Car, Model, Year, Price, Location, etc.**)

#### 💬 **Messaging & Notifications**
- In-App Messaging System
- Email & Push Notifications (**New Listings, Bids, Messages, Promotions, etc.**)

#### 💳 **Payment & Transactions**
- Secure Payment Gateway
- Transaction History & Invoicing
- Escrow System for Secure Transactions

#### 🛠 **Admin Dashboard**
- User & Listing Management
- Ad & Payment Management
- KYC Approval System
- Content Moderation (**Reports, Bans, Warnings**)

---

## ⚙️ Technology Stack

### 🎨 **Frontend**
- **Framework:** React (with Vite)
- **State Management:** Redux/Zustand
- **UI Library:** Tailwind CSS / Material UI
- **API Calls:** Axios / Fetch API

### 🖥 **Backend**
- **Framework:** Express.js
- **Database:** MySQL (Relational Data)
- **Authentication:** JWT, OAuth2 (OneSignal)
- **Storage:** Backblaze for Images/Documents
- **Payment Integration:** Stripe, PayPal, APIs

### 🌍 **Other Services**
- **Geolocation API:** Google Maps / OpenStreetMap
- **WebSockets:** Real-Time Messaging
- **Notification Services:** Twilio, SendGrid

## 🚀 Get Started
1. Clone the repository:
   ```sh
   git clone https://github.com/Debayan-Sarkar/Barablier-development.git
   ```
2. Install dependencies:
   ```sh
   cd barabiler
   npm install  # For frontend
   ```
3. Run the project:
   ```sh
   npm run dev  # Start frontend development server
   ```
4. Backend Setup:
   - Configure **Node.js (Express.js)** backend.
   - Set up **MySQL** database.
   - Integrate API keys for **payments, notifications, and geolocation**.
