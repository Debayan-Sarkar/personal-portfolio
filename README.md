1. Introduction
Barabilar is an online platform that enables individuals and businesses to buy, sell, swap, and auction cars. The platform also supports businesses offering car-related services, including spare parts sales. Additionally, businesses can run paid advertisements to promote their listings. Businesses need to pay 50 kron per post.

3. Technology Stack
Frontend
Framework: React (with Vite)
State Management: Redux/Zustand
UI Library: Tailwind CSS or Material UI
API Calls: Axios/Fetch API
Backend
Framework: PHP Laravel (Express.js)
Database: MySQL (for relational data)
Authentication: JWT, OAuth2 (OneSignal)
Storage: Backblaze for Images/Documents
Payment Integration: Stripe, PayPal, APIs
Other Services
Geolocation API (Google Maps, OpenStreetMap)
WebSockets for Real-Time Messaging
Notification Services ( Twilio, SendGrid)
